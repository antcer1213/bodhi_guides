Terminology


About Terminology

Terminology is the default Bodhi Linux terminal emulator, accessable via Main Menu>Applications>System Tools. It is a very new terminal emulator based entirely on Enlightenment. In fact, the first day of its creation was the 12th of June 2012.

The purpose of Terminology, like all terminal emulators, is to give you access to the command-line interface, or CLI. It does this task perfectly, and the CLI is a very powerful tool, if you learn how to use it! For those with limited knowledge on the CLI, anything you can do with applications featuring graphical user interfaces (GUIs) can be done from the command-line, and then some. For people who have little experience with the CLI, it's generally easier to do activities with a GUI. However, the command line is much quicker, more convenient, and familiarizing yourself with it is highly recommended.

Note: A how-to or tutorial on the CLI is too broad in scope to be covered in the Guide. Might we suggest a Google search? ;)

The way you interact with Terminology is rather straightforward. For example, when you wish to copy/paste a string of text into Terminology, it is as simple as a right-click. A menu then pops out on the right side of the Terminology window displaying the easily understood controls Copy, Paste, Options, and About. In fact, right-click is used to initiate many of the interactions with Terminology: right-clicking anywhere in the Terminology window opens the same pop-out menu at the right. You can change many settings for Terminology using the Options button.

Now for the things that set it apart. Terminology is a powerhouse of ability and style.

  • It can have a background of either a bitmap(ex. jpg, png, bmp), a scalable/vector(ex. svg, pdf), an animated gif, or a video. Yes, you read that correctly. A video! To assign any of these, run this simple command in Terminology or, for a permanent method, modify its launcher (as described here) to read:

    terminology -b /path/to/file

  • It can change fonts and font-sizes easily. It has bitmap fonts support, by default.


As the E17 developers said:

If you like a terminal that scrolls more than 10 times faster than gnome-terminal, much faster than Xterm or Konsole and as fast as URXVT, then this is for you. If you like to save memory compared to the big-desktop terminals (gnome-terminal and Konsole) then this is for you.

Adding Another Terminal Emulator

For most, Terminology will meet all your needs for a terminal emulator. For those of whom it does not, many others are available. And, some people just have their favorite terminal emulators! Therefore, we will go through an example for installing an additional terminal emulator.

Installing an additional terminal emulator is very easy. For the example, we will install LXTerminal, a light but powerful terminal emulator app that was formerly the default for Bodhi Linux.

Note: Several alternative terminal emulators are available for installation, though we particularly recommend the terminal emulators available at Bodhi's Appcenter.

To install LXTerminal, open Terminology via Main Menu>Applications>System Tools. Enter the following command:

sudo apt-get install -y lxterminal

Terminology will ask you for your password; type it in and hit Enter. After the installation finishes, do a quick Main Menu>Enlightenment>Restart to refresh the desktop. Now you should see LXTerminal in your menu at: Main Menu>Applications>Accessories. If you'd like, you can add LXTerminal to your Favorites menu, as explained here.

Now you have two terminal emulators. You can use one, both, or install another, and weight your options!

Enlightenment File Manager (EFM)


About EFM

The Enlightenment File Manager (EFM), which is a native Enlightenment app, is the default Bodhi Linux file manager. You can access it via Main Menu>Applications>System Tools. By default, EFM will show your Home drive, the Desktop, the Root folder system, and any mounted volumes or drives. Hovering over a folder or file in the window will open a pop-up (called a tooltip) that gives information about the folder/file.

EFM gives you multiple ways to move through the file system. As described here, there is the menu-integrated form of EFM known as Navigate. In the application, itself, it has up, down, and back arrows to allow you to move one step at a time from the folder open in the window to another, or you can open any of the folders or files present to navigate.

As with most things in Enlightenment, you can modify how EFM looks and works in multiple ways. The most direct is to right-click in an empty space in the folder/file-pane, which will open a settings menu. You can also access the EFM settings via:

Main Menu>Settings>All>Files>File Manager
or
Main Menu>Settings>Settings Panel, under Files>File Manager


Now for the things that set it apart. EFM is a powerhouse of ability, speed, and style.

  • It is fully customizable via themes. You can literally make it look exactly like you want. For a general introduction to creating themes, go to the Creating Themes page.
  • It can assign individual directories a bitmap background (ex. jpg, png, bmp). To assign a bitmap, go to the directory you wish to assign a background. Then right-click an empty area of EFM and go to View Mode> Set background. From there, it is as easy as selecting a bitmap and clicking OK.
  • It has the ability to display desktop icons. To enable this, open EFM and right-click on an empty area. From there, go to Options> File Manager Settings. In the Fileman Settings dialog that pops up, navigate to the Display tab and tick "Icons on Desktop". Now, simply copy or move any files (application launchers, a.k.a .desktop files, included) you wish to display on your desktop to the Desktop folder EFM creates in your home directory.
  • Its toolbar serves as a shelf of sorts, allowing you to add/remove certain gadgets from/to it by right-clicking it and going to toolbar>toolbar contents.
  • Its tooltip can preview video files! Yes, you read that correctly. It can preview video files.

Adding another File Manager

Although EFM is a fully capable file manager, you may decide you want something different for some reason or other. No problem; Bodhi Linux encourages you to make your own choices as regards to software.

Installing a new file manager is very easy. As you may have guessed, there is more than one way to install another file manager. For a detailed list of the various methods, see the Installing Software page of the Bodhi QuickStart Guide. For this example, we will use the apt-get command to install PCManFM, a light but powerful file manager that was formerly the default of Bodhi Linux.

Note: Several alternative file managers are available for installation, though we particularly recommend the file managers available at Bodhi's Appcenter.

To install PCManFM, open Terminology via Main Menu>Applications>System Tools. Then enter the following command:

sudo apt-get install -y pcmanfm

Terminology will ask you for your password; type it in and hit Enter. After the installation finishes, do a quick Main Menu>Enlightenment>Restart to refresh the desktop. Now you should see PCManFM in your menu at: Main Menu>Applications>Accessories.

Enjoy the options!
