Bodhi Guide - General Settings


Enlightenment (E17) Settings in Bodhi Linux can be accessed via the Settings Panel at Main Menu -> Settings -> Settings Panel. Each of the settings dialogs can also be accessed directly under Main Menu -> Settings -> All.

Note: Some settings options are only available when certain Modules are loaded.

Setting

Description

Look

Wallpaper
Change the wallpaper for all desktops or current desktop.
Application Theme
Options for themes and GTK-icons for applications.
Theme
Switch Enlightenment themes. Advanced options allow assigning Themes to individual elements.
Colors
Assign custom colors to individual Enlightenment elements. Colors depend on Theme capabilities.
Fonts
Assign custom font preferences for Enlightenment. Advanced options allow assigning customizations to individual elements.
Borders
Set default window border style.
Transitions
Set the type of transition to use for certain events, such as changing the background.
Scaling
Scale the overall font settings throughout Enlightenment.
Startup
Set the splash screen animation to use at boot-up.
Dropshadow
Settings for dropshadow effects on the desktop from windows and other objects (Dropshadow module must be loaded).

Apps

Personal Application Launchers
GUI for creating .desktop files which can be added to the Main Menu.
Favorite Applications
Add/Remove entries to the Favorites menu. Also allows changing order of appearance.
iBar Applications
Add/Remove applications to the IBar Gadget. Also allows changing the order of appearance.
Screen Lock Applications
Select applications to automatically run when screen is locked.
Screen Unlock Applications
Select applications to automatically run when screen is unlocked.
Restart Applications
Select applications to automatically run after restarting E17.
Startup Applications
Select applications to automatically run at startup.
Default Applications
Select the default application for certain categories of apps, such as browser.
Desktop Environments
Set options for X, and enable options from other desktop environments.

Screen

Virtual Desktops
Adjust the number and layout of Virtual Desktops as well as the animation when flipping between them.
Screen Resolution
Set the screen resolution.
Screen Lock
Set screen lock time as well as login box location, wallpaper, etc.
Blanking
Enable/Disable X Screensaver, set timer settings, etc.
Backlight
Set backlight options where available.

Input

Edge Bindings
Set actions to occur when using the mouse to interact with the edges of the screen.
Key Bindings
Keyboard shortcut settings.
Mouse Bindings
Set actions to bind to mouse buttons. Mouse buttons can be combined with keyboard for additional bindings.
ACPI Bindings
Set actions to specialized hardware bindings such as pressing the power button or closing the lid to your laptop.
Touch
Settings for touchpads.
Mouse
Choose between X or Enlightenment style mouse pointer as well as set the size, and select mouse button orientation and mouse acceleration speed.

Windows

Window Display
Set window display behavior including moving, resizing, new windows, shading, and screen limits.
Window Focus
Adjust how mouse interaction changes window focus. Advanced options available, including window stacking and hinting.
Window Geometry
Adjust window resistance to elements on the screen and parameters when moving/resizing windows via the keyboard. Set behavior when maximizing a window. Options include Fullscreen, Smart Expansion, Fill available space. Can also set direction of window expansion.
Window List Menu
The Window List allows cycling between windows, and is typically accessed by pressing Alt+Tab. The menu lets you set window grouping behavior, sort order, and association of iconified windows with virtual desktops. Also can limit length of captions for iconified windows.
Window Remembers
Automatically use most recent window settings for internal dialog or file manager windows next time they are opened. Can remember all or a subset of window settings including size, position, locks, and more. This dialog is primarily to delete window remembers. Window remembers are set using the window menu.
Window Switcher
Set how open windows are listed in the Window List Menu, which can be accessed either through middle-click of a three-button mouse or via Main Menu -> Windows.

Menus

Menu Settings
Enlightenment Main Menu settings. Adjust what application info is displayed as well as menu scrolling behavior.

Language

Desklock Language Settings
Set a different language for lock menu.
Language Settings
Set the locale settings for the environment.
Input Method Settings
Adjust settings for various Input Methods which allow users to input symbols not found on their input device. For example, entering Chinese characters with a Latin keyboard.

Advanced

Signal Bindings
Set actions to be taken for mouse clicks.
Syscon
Adjust icon sizes in System Menu (opened by clicking on System in Main Menu).
Performance
Fine tune E17 performance. Adjust framerate, font cache, image cache, edje cache, etc.
Engine
Enable use of ARGB windows instead of shaped windows.
Power Management
Time to delay power-hungry tasks.
Search Directories
View/Set directory search paths E17 uses to locate various files, such as Images, Fonts, Icons, etc.
Battery Meter
Control how battery is monitored. Set polling time, low battery alert criteria, etc.
Configuration Panel
Enable/Disable E17 configuration dialogs to be accessed from Main Menu -> Settings -> All.
Environment Variables
Set custom E17 environment variables.

Launcher

Everything Configuration
Configure the Run Everything program including how choices are sorted, view type, size, alignment, and active plugins.
Everything Plugin: Applications
Configure terminal and GUI commands for starting applications.
Everything Plugin: Files
Set whether to search and/or show recent files or cache. Also provides option to clear cache.
Everything Plugin: Plugins
General configurations for plugins to Run Everything.
Everything Plugin: Start
Set order of Everything plugins.

Settings

Dialogs
Set options for dialog windows including if they should behave like normal windows and, if available, whether they should automatically open in Basic or Advanced mode.
Profiles
Create, delete, and switch between different Profiles.

Extensions

Modules
Load/Unload Enlightenment modules.
Shelves
Add/Delete Shelves and configure settings for individual Shelves.
Mixer
Set which mixer to use for sound and open Mixer channel settings.
Notification
Select what notifications appear, and how they are displayed.
Pager
Configure settings for Pager Gadgets.
Gadgets
Add available Gadgets to the desktop.

Files

File Manager
Adjust settings for the Enlightenment File Manager (EFM).
File Icons
Set icons used by EFM for different file types.
Places
Adjust settings for Places module (appears only if the Places modules is loaded). Allows access to file manager via Main Menu as well as using a custom file manager (e.g. PCManFM).

System

**These settings require root privilges
Login Configuration
Setup user for automatic login and more.
Elementary Configuration
Set options for widgets that use the Elementary widget set.
Synaptic Package Manager
Install, uninstall, download, or update packages from available repositories.
Synaptic Package Manager
Install, uninstall, download, or update packages from available repositories.
